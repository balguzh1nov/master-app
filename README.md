# Мебельщики — Поиск мастеров

Приложение для поиска мебельщиков и создания заявок на выполнение работ.

## Технологии

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- React 19

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для production
npm run build

# Запуск production сервера
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

## Структура страниц

### Для клиентов:
- `/` - Главная страница с поиском мебельщиков
- `/master/[id]` - Профиль мебельщика
- `/create-request/[masterId]` - Создание заявки
- `/chat/[id]` - Чат с мебельщиком
- `/reviews/[id]` - Отзывы о мебельщике

### Для мебельщиков:
- `/requests` - Список заявок от клиентов
- `/profile` - Профиль мебельщика (редактирование)
